## 3. 日期覆盖与衔接（风光两套）

- **风光两类存在两套口径**：
  - **带「-汇总」**（光伏-汇总、风力-汇总）：2026-01-01～**2026-02-02**（33 天），单列「实际发布电力（MW）」。
  - **无「-汇总」**（光伏、风力）：**2026-02-03**～2026-02-28（26 天），列含江南/江北/汇总三列。
- 两套在日期上**衔接、不重叠**：1 月 1 日～2 月 2 日用带-汇总；2 月 3 日起用无-汇总。**无同一天同时存在两类文件**，无法在同一时刻逐点对比两类中的「汇总」数值。
- **无-汇总文件中的「汇总」列**：对 2026-02-03 的光伏、风力文件逐行核对，**汇总列 = 江南 + 江北**（浮点误差内一致），即无-汇总文件里的「汇总」为江南、江北之和。带-汇总文件里的「实际发布电力」从口径上应为全省汇总，但因无重叠日无法在本数据中直接验证与无-汇总文件中「汇总」列是否一致。

## 4. 文件结构

### 4.1 总体

- 所有文件均为 **1 个 sheet，sheet 名均为 `sheet`**。
- 列结构分两种：**单数值列**（3 列）或 **江南/江北/汇总三数值列**（5 列）。

### 4.2 单数值列（3 列：序号 + 时间 + 1 个 MW 列）

| 文件类型 | 数值列名 |
|----------|----------|
| 实际受电情况-华东 | 实际发布电力（MW） |
| 实际燃机固定出力总值-汇总 | 实际总值（MW） |
| 实际系统负荷 | 实际发布电力（MW） |
| 储能发电计划实际数据-汇总 | 实际数据（MW） |
| 煤电发电计划实际数据-汇总 | 实际数据（MW） |
| 实际统调风光情况-光伏-汇总 | 实际发布电力（MW） |
| 实际统调风光情况-风力-汇总 | 实际发布电力（MW） |

### 4.3 三数值列（5 列：序号 + 时间 + 江南/江北/汇总）

仅**文件名中无「-汇总」**的光伏、风力文件（即「光伏」「风力」后直接跟日期），自 2026-02-03 起，共 26 天。  
列名：`序号`、`时间`、`江南实际发布电力（MW）`、`江北实际发布电力（MW）`、`汇总实际发布电力（MW）`。

- 要看江南/江北/汇总三列，请打开如 **实际统调风光情况-光伏-2026-02-03.xls**（中间无「-汇总」）；若打开 **实际统调风光情况-光伏-汇总-2026-01-01.xls** 则只有 3 列。
- **序号**：build_dataset 默认排除；**时间**：表内存在，96 点可解析；单位均为 **MW**。
