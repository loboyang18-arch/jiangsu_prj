# report 目录说明

本目录存放**方案与报告**，整理后仅保留当前主流程所需文档。

## 核心文档

### 数据与入库

| 文件 | 说明 |
|------|------|
| **数据分析报告.md** | 总表与江苏数据来源、行级/值级缺失汇总、特征可用性与缺口处理结论 |
| **总表数据入库方案.md** | 入库原则、库表设计、样本窗（含优先主窗 2025-09-01～12-31）、缺失修复、版本 V0/V1/V2/V3 |
| **总表入库审计报告.md** | 入库审计结果：覆盖率、质量标记分布、V0 特征剔除验证 |
| **总表入库代码审查.md** | Stage/DWD/V0 代码实现审查、V1 预测链路与发布时间规范 |
| **子文件夹梳理/** | 01～09：江苏/总表各子目录的逐文件梳理（行缺、值缺、有效起止） |

### 日前价格预测

| 文件 | 说明 |
|------|------|
| **日前价格预测技术报告.md** | 完整技术研究报告：问题定义、特征工程、模型架构（V2-V7）、后处理、实验分析 |
| **日前价格预测技术报告（简版）.md** | 面向非技术读者的缩略版 |
| **日前价格预测模型迭代报告.md** | V2→V7 全版本对比、消融实验、逐折详情、最优配置与命令行 |
| **外生变量相关性分析.md** | D 日外生变量（clearing/actual）与价格的相关性分析、Oracle 实验验证（负结论） |

### 辅助分析
