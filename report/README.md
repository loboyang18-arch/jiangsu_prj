# report 目录说明

本目录存放**方案与报告**，整理后仅保留当前主流程所需文档。

## 核心文档（保留）

| 文件 | 说明 |
|------|------|
| **总表数据入库方案.md** | 入库原则、库表设计、样本窗（含优先主窗 2025-09-01～12-31）、缺失修复、版本 V0/V1/V2/V3 |
| **数据分析报告.md** | 总表与江苏数据来源、行级/值级缺失汇总、特征可用性与缺口处理结论 |
| **总表入库审计报告.md** | 入库审计结果与文件主题-列名对应 |
| **日前价格预测模型迭代报告.md** | 日前价格预测 V2→V5 全版本对比、消融实验、最优配置与逐折分析 |
| **子文件夹梳理/** | 01～09：江苏/总表各子目录的逐文件梳理（行缺、值缺、有效起止） |

## 可复现产出（已删除，按需运行脚本生成）

- 字段字典：`data_dictionary.md` / `.csv` → `python scripts/scan_excel_dictionary.py ...`
- 原始数据质量报告：`raw_data_quality_*` → `python scripts/raw_data_quality_audit.py ...`

详见项目根目录 [README.md](../README.md) 的「整理说明」与「使用说明」。
