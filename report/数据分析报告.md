# 总表与江苏数据分析报告（正式发布稿）

## 版本说明

本报告基于对「总表」与「江苏」两个数据来源下各子文件夹与单文件的**逐项梳理结果**（`report/子文件夹梳理/01～09`）形成，汇总两类来源在**数据结构、业务口径差异、数据质量（行级/值级缺失）、特征可用性与建库选源策略**上的主要结论，用于支持后续数据建库、特征工程设计、模型训练及数据采购决策。

- **对应基础文档**：`report/子文件夹梳理/01～09`。  
- **适用范围**：江苏电力市场相关价格、负荷、受电、燃机、煤电、储能、风光、备用等 **15 分钟粒度（96 点/日）** 数据的统一整理与建模准备。  
- **可追溯性约定**：本报告引用的**文件名、sheet 名、列名均使用原文全称**（含括号、分号等字符），便于读者定位源表。

### 1.3 时间粒度与统一时间轴（timestamp）说明

两类来源均以 **96 点/日** 的 15 分钟粒度为主（通常首点 00:15、末点 24:00）。建库与建模前必须统一构造完整时间戳（timestamp）：  

- **总表**：通常可由表内 `查询日期` + `时间` 直接拼接得到 timestamp。  
- **江苏**：多数文件为「文件名含日期 + 表内含 `时间` 列」结构，应以文件名日期 + 表内 `时间` 构造 timestamp；少数文件（如无时间列的 sheet 或宽表）需按结构推断并明确记录规则。  
- **行序注意事项**：总表中同一天的 96 行记录**不保证按时间自然排序**；抽取或取样必须先按 `查询日期 + 时间` 排序，或按时间值精确定位。  

时间范围上，总表整体覆盖约 **2025-06-01～2026-01-26**；江苏目录中 2025 年数据主要覆盖 **2025-06～2025-12**，2026 年数据主要覆盖 **2026-01～2026-02**，部分目录存在连续或零星缺口（见第 3 节）。

## 2. 两份数据的异同（结构、口径与重叠关系）

### 2.1 文件组织形态与业务口径差异

| 维度 | 总表 | 江苏 |
|------|------|------|
| **文件组织形态** | 合并总表：一主题对应一张（或少数）总表，多日合并在一文件 | 按日拆分为主：一日一文件或一类一文件 |
| **时间字段表达** | 通常直接提供 `查询日期` + `时间` | 多为「文件名日期 + 表内 `时间`」，少数需从结构推断 |
| **阶段划分** | 事前 / 事后、汇总 / 江南 / 江北划分清晰 | 需依靠文件名、列名及内容判断“计划/预测/实际/公开” |
| **命名风格** | 偏业务总表式命名（例：`受电计划_合并总表.xlsx`、`实际电网运行情况_江北分区_合并总表.xlsx`） | 偏运行数据式命名（例：`实际受电情况-华东-YYYY-MM-DD.xls`、`实时出清结果统合-YYYY-MM-DD.xlsx`） |

**一致**：指在重叠日期、同一时刻逐点相等（浮点误差内），可初步视为同源或可替代来源。正式建库前建议补充一次全量一致性校验并固化为字段映射表。

|----------|------------------------------------------|-------------------------------------------|----------|
| **实际受电（MW）** | `总表/事后/事后_汇总总表合集/实际受电情况_合并总表.xlsx` / `实际受电情况数据` / `实际发布电力（MW）` | `江苏/电价数据/25年电网边界/实际受电情况-华东-YYYY-MM-DD.xls` / `sheet` / `实际发布电力（MW）` | **一致** |

| **光伏/风力/储能/煤电/燃机（事后汇总，MW）** | `总表/事后/事后_江北总表合集/*.xlsx` + `总表/事后/事后_江南总表合集/*.xlsx` **分区相加得到汇总** | 江苏对应的汇总日表（见 2.3） | **一致** |

### 2.3 “事后分区相加 = 江苏汇总”的字段级对应（已抽样验证）

- **光伏**：总表 江北+江南 光伏 `实际发布电力（MW）` = 江苏 `实际统调风光情况-光伏-汇总-YYYY-MM-DD.xls` 同名列，**一致**。  
- **风力**：总表 江北统调风光（风力）+ 江南风力 `实际发布电力（MW）` = 江苏 `实际统调风光情况-风力-汇总-YYYY-MM-DD.xls` 同名列，**一致**。  
- **储能**：总表 江北+江南 储能 `实际数据（MW）` = 江苏 26年 `储能发电计划实际数据-汇总-YYYY-MM-DD.xls` 同名列，**一致**。  
- **煤电**：总表 江北+江南 煤电 `实际数据（MW）` = 江苏 26年 `煤电发电计划实际数据-汇总-YYYY-MM-DD.xls` 同名列，**一致**。  
- **燃机**：总表「实际电网运行情况」江北+江南 `实际总值（MW）` = 江苏「实际燃机固定出力总值-汇总」`实际总值（MW）`，**一致**（命名不同、口径相同）。  

### 2.4 独有与互补信息

- **总表独有**：`正负备用空间_合并总表.xlsx`（`正备用(MW)`、`负备用(MW)`）；事前受电计划、短期负荷预测、事前江北/江南储能/煤电/燃机/风光计划与预测。  
- **江苏独有或延长时段**：`26年日前电价.xlsx`（2026-01-01～02-28 宽表）；`26年实时价格/实时市场加权均价（公开）--YYYY-MM-DD.xls`（2026-01-01～02-28，缺 2026-01-06）；`26年电网实际边界/*.xls`（2026-01-01～02-28）。

### 3.3 格式、列名与结构风险
