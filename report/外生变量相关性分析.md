# D 日外生变量与日前出清价格的相关性分析

> 分析日期: 2026-03-16
> 数据: 122 个交易日 × 96 时段 = 11,712 条记录
> 目的: 评估 D 日的出清值(clearing)和实际值(actual)是否与日前出清价格强相关，以及是否值得"先预测外生变量，再用于价格预测"

---

## 一、背景

当前价格预测模型使用 D 日的 **boundary（边界/预测值）** 作为特征——这些在 D-1 09:00 就发布了。但数据中还存在两类 D 日变量未被利用：

- **Clearing（出清值）**：日前市场出清后确定的负荷/新能源/气电等调度计划，D-1 ~17:00 随出清价格一同发布。
- **Actual（实际值）**：D 日实际运行的负荷、风电、光伏、气电出力等，D+1 才能拿到。

核心问题：**这些变量与价格的相关性是否显著强于 boundary？如果是，能否通过先预测它们来间接提升价格预测？**

### 2.3 相关性排名关键发现

1. **净负荷是与价格相关性最强的外生变量**（Spearman 0.40-0.43），远超其他单一变量。净负荷 = 负荷 - 新能源出力，反映火电需求强度。

2. **actual > clearing > boundary**，但差距不大：
   - 净负荷：actual 0.420 vs boundary 0.398，差距仅 0.022
   - 负荷：actual 0.377 vs boundary 0.347，差距 0.030
   - 新能源渗透率：三者几乎一样（0.309-0.311）

3. **光伏与价格的相关性出乎意料地弱**（|Spearman| < 0.09）。原因：光伏出力白天高、夜晚零，但价格在夜晚也不高（因为夜晚需求也低），两者的日内变化模式正负抵消。

4. **气电（江北）的 clearing 与 actual 相关性显著高于 boundary**（clearing 0.225 vs boundary 0.186），气电是唯一"出清值比边界值有明显增量信息"的变量。

### 关键结论

1. **风电和光伏的 clearing 与 boundary 几乎一样**（差异 < 1%）。新能源的出清值基本等于预测值——出清过程不改变新能源的调度计划。**这两类的 clearing 没有预测价值。**

2. **负荷的 clearing 与 boundary 差异也很小**（0.7%）。而且差异与价格几乎不相关（Spearman < 0.03）。**负荷 clearing 也没有增量价值。**

3. **气电的 clearing 与 boundary 差异较大**（7-8%），且差异与价格有微弱正相关（0.03-0.07）。气电出力是出清优化的调节变量——价格高时气电多发、价格低时气电少发——所以 clearing 比 boundary 多反映了市场供需信息。**但相关系数太弱（<0.08），实际增量有限。**

4. **光伏(江北)的差异虽小但与价格有一定相关（-0.114/-0.135）**。这可能反映了出清过程中对光伏消纳的微调与价格之间的关联。

## 四、Actual vs Boundary：实际值与预测值的偏差分析

| 变量 | Boundary MAE | 相对误差(MAPE) | 偏差与JN价格Spearman | 偏差与JB价格Spearman |
|------|----------:|--------:|-------------------:|-------------------:|
| **负荷** | 2,374 | **2.7%** | **+0.100** | **+0.176** |
| 受电(华东) | 817 | 4.6% | -0.009 | +0.004 |
| **光伏(江北)** | 336 | **10.2%** | -0.026 | +0.058 |
| **光伏(江南)** | 77 | **12.9%** | +0.046 | **+0.107** |
| **风电(江北)** | 1,304 | **20.8%** | +0.038 | +0.032 |
| 风电(江南) | 13 | 37.5% | +0.018 | -0.016 |
| **气电(江北)** | 138 | **14.2%** | **+0.090** | **+0.105** |
| **气电(江南)** | 526 | **13.4%** | **+0.096** | **+0.113** |

### 关键结论

1. **负荷偏差与价格有最强相关性**（JB: +0.176）：实际负荷比预测高 → 价格偏高。这是物理上直觉的——负荷超预期意味着供需偏紧。

2. **气电偏差与价格正相关**（0.09-0.11）：实际气电出力 > 计划 → 价格偏高。这也合理——气电是灵活调峰电源，价格高时调度增发气电。

3. **光伏(江南)偏差与JB价格正相关**（+0.107）：实际光伏 > 预测 → 价格偏高？这有些反直觉，可能是因为光伏偏差与负荷偏差存在共变关系。

4. **风电偏差最大（MAPE 20-37%）但与价格几乎不相关**（<0.04）。说明风电的随机性是"噪声"而非"信号"——它的预测误差不系统性地影响价格方向。

5. **受电(华东)偏差与价格完全不相关**（Spearman ≈ 0）。省际受电的实际值与计划值的偏差对价格没有影响。

| 受电(华东) | **0.976** | 3.9% | **0.972** | 4.6% |
| 光伏(江北) | **1.000** | 0.3% | 0.963 | **10.2%** |

| 风电(江北) | **1.000** | 0.2% | 0.944 | **20.8%** |
| 风电(江南) | 0.994 | 0.6% | 0.862 | **37.5%** |
| **气电(江北)** | **0.782** | **7.5%** | **0.664** | **14.2%** |
| **气电(江南)** | **0.890** | **8.3%** | **0.826** | **13.4%** |

### 关键结论

1. **风电和光伏的 clearing ≈ boundary**（Pearson > 0.99，MAPE < 1%）。出清过程几乎不改变新能源调度，预测 clearing 毫无意义。

2. **风电和光伏的 actual 与 boundary 有显著偏差**（MAPE 10-37%），但偏差与价格相关性极弱。即使能精准预测实际风光出力，对价格预测的增量也很小。

3. **气电是唯一 boundary 与 clearing/actual 差异较大且与价格相关的变量**。但 boundary→clearing 的 Pearson 仅 0.78-0.89，说明气电出清值的可预测性本身就有限。

4. **负荷的 boundary→actual 精度很高**（Pearson 0.955, MAPE 2.7%），且偏差与价格有一定相关性（0.10-0.18）。这是最有价值的"先预测再使用"的候选变量。

## 六、综合结论与建议

### 6.1 值得追求的方向

| 优先级 | 目标变量 | 理由 | 预期收益 |
|--------|---------|------|---------|
| **高** | **D日实际负荷** | 与价格相关性最强的外生变量之一（偏差 Spearman 0.10-0.18），boundary 预测精度已较高（MAPE 2.7%），有改善空间 | 如果能将负荷预测 MAPE 从 2.7% 降至 1.5%，净负荷精度的提升可能带来可观的价格预测改善 |
| **中** | **D日气电出力** | clearing/actual 与 boundary 差异较大（7-15%），且差异与价格正相关（0.06-0.11）。但 boundary→actual 的 Pearson 仅 0.66-0.83，本身预测难度大 | 增量有限但非零，适合作为辅助特征 |
| **低** | D日光伏/风电出力 | 虽然 actual 与 boundary 偏差大，但偏差与价格几乎不相关。clearing ≈ boundary，无增量 | 不建议投入 |
| **不建议** | D日受电(华东) | 偏差与价格完全不相关 | 无价值 |

### 6.2 推荐的两阶段预测管道

```
阶段 1: 预测 D 日实际净负荷
  输入: D 日 boundary（负荷预测、新能源预测）+ D-1 的负荷/新能源偏差模式 + 日历特征
  目标: D 日实际负荷（或直接预测净负荷）
  价值: 净负荷 actual 与价格的 Spearman=0.42-0.43，比 boundary 的 0.40 高 5%

阶段 2: 用预测的净负荷替代 boundary 净负荷，输入价格预测模型
```

### 6.3 一个重要的现实约束

actual vs boundary 的相关性差距虽然存在但不大（Spearman 0.42 vs 0.40）。这意味着：

- **如果净负荷预测不能比 boundary 更准**，那引入预测值反而会增加噪声
- 只有当我们能利用**boundary 之外的额外信息**（如 D-1 的实际运行偏差模式、气象数据等）来修正 boundary 的负荷预测时，这条路才有价值
- 本质上，这等价于在说"调度中心的负荷预测不够准，我们能做得更好"——这是一个需要验证的假设

## 七、Oracle 实验验证（2026-03-16）

### 7.1 实验设计

为量化"完美预测 D 日外生变量"的理论价格提升上限，我们进行了 **Oracle 实验**：直接将 D 日 actual（实际值）作为特征输入价格模型，跳过"预测外生变量"这一步。这代表了"即使能 100% 准确预测 D 日外生变量"的理论最优情况。

**实验配置**：
- **基线**：V5 各自最优配置（JN: quantile-two-stage + ANB + DFV; JB: two-stage + 时间衰减 + ANB + DFV）
- **Oracle A**：基线 + 16 个 D 日 actual 特征（actual 负荷、风电、光伏、气电、受电、净负荷、新能源渗透率等）
- **Oracle B**：基线 + 仅 3 个 actual 净负荷特征（actual_net_load, actual_net_load_day_mean, actual_minus_bnd_net_load）
- CV 框架：7 折 expanding-window，完全一致

### 7.2 实验结果

| 配置 | JN MAE | JN RMSE | JB MAE | JB RMSE |
|------|-------:|--------:|-------:|--------:|
| **V5 基线** | **78.20** | **117.85** | **69.61** | **110.36** |
| Oracle A（16 actual 特征） | 83.54 | 122.94 | 69.87 | 111.74 |
| Oracle B（3 净负荷特征） | ~88.6 | ~118.0 | ~70.7 | ~113.9 |

| 配置 | JN MAE Δ | JB MAE Δ |
|------|:--------:|:--------:|
| Oracle A vs 基线 | **+6.8% (更差)** | **+0.4% (更差)** |
| Oracle B vs 基线 | **~+0% (持平)** | **+1.6% (更差)** |

### 7.4 关键结论

1. **Oracle 实验结果为负**：即使给模型 D 日 100% 准确的 actual 外生变量（作弊级别），价格预测不仅没改善，反而变差了。这一结论在全量 16 特征和精简 3 特征两种配置下均成立。

2. **变差的原因分析**：
   - **特征冗余导致过拟合**：actual 变量与 boundary 变量高度共线性（Pearson 0.86-1.00），加入后增加模型复杂度但未带来真正的新信息
   - **样本量不足**：122 天的训练数据无法支撑更多特征的有效学习
   - **概念漂移放大**：后期折（Fold 5-6）恶化最严重，说明 actual 特征的模式在短数据窗口内不稳定

3. **对"两阶段预测管道"的否定**：既然 100% 准确的 actual 净负荷都无法改善价格预测，那么"先预测净负荷再用于价格预测"这条路线**在当前数据量和模型框架下没有价值**。

4. **根本原因**：相关性分析中 actual 净负荷的 Spearman r=0.42 vs boundary 净负荷的 r=0.40，差距仅 0.02。这个微弱的增量信息量远不足以在 122 天的小样本中产生统计显著的预测改善。boundary 已经捕获了净负荷与价格关系的绝大部分信息。

### 7.5 后续建议

基于 oracle 实验的负结论，建议：

- **放弃**"预测 D 日外生变量 → 间接改善价格预测"的技术路线
- **价格预测的改进空间更可能在于**：
  - 积累更长时间的训练数据（当前仅 4 个月）
  - 引入与价格直接相关但当前未使用的信息源（如市场报价行为、跨省交易计划等非物理量特征）
  - 改善地板价/尖峰价的建模能力（当前模型的主要瓶颈）
